# Gilroy font files

Place these three font files in this folder:

- **Gilroy-Regular.ttf** — weight 400 (body text, captions)
- **Gilroy-Medium.ttf** — weight 500 (labels, secondary text)
- **Gilroy-SemiBold.ttf** — weight 600 (titles, headings, buttons)

The app uses only these three weights for consistent UI/UX. Get the font from your design team or a licensed source. If you use a different file naming, update the paths in `pubspec.yaml` under `flutter.fonts`.
